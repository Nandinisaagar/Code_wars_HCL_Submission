# Code_wars_HCL_Submission
### Health Care provider website
#### - Deliverables:
1. Develop a solution for a Healthcare Wellness and Preventive Care Portal that  seamlessly integrate the frontend with the backend technologies.
2. Portal should be focused on wellness and preventive care to help paitents achieve basic health goals and maintain compliance with preventive checkups.
3. Portal should prioritize usuability and key preventive care features.
4. Special attention should be given to responsive design, authentication, profile management and compliance with healthcare privacy standards.

#### - Solutions:
1. We built a website that is providing medical services and enabling paitents to connect with the doctors reviewing their medical reports and booking appointments all at one stop.
2. In the website we have integrated a wellness dashboard for the paitents tracking their health and updating with next doctor's appointment.
3. Yes, Portal is highlighting the usuablity and priotizing health.
4. Yes, design, authentication and authorization is implemented efficiently.
